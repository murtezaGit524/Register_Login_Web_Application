Welcome to My Project!

Project Description

This is a web-based registration and login system implemented using HTML, CSS, Java Servlet, JSP (Java Server Page),
and MySQL. It allows users to register by providing necessary information and subsequently login to access secured features.
The combination of front-end technologies HTML and CSS, while java servlet handle server-side logic and JSP facilitates dynamic
content generation. The data entered during registration is stored securely in a MySQL database, enabling persistent user information.
